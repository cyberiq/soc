## 🎯 الميزات

### 1. تتبع الـ IPs الفريدة
- كل IP جديد يُعتبر زائر جديد
- التطبيق يتذكر جميع الـ IPs
- يمكن الوصول إليها عبر `/api/visitors`

### 2. اكتشاف الأجهزة
- **نوع الجهاز:** Desktop / Mobile
- **نظام التشغيل:** Windows, macOS, Linux, Android, iOS
- **المتصفح:** Chrome, Firefox, Safari, Edge, Opera, IE
- **اللغة:** من Header Accept-Language
- **User-Agent:** كامل معلومات المتصفح

### 3. تتبع المصدر
- **الرابط المطلوب:** (path)
- **المرجع:** من أين جاء الزائر
- **الوقت:** وقت الوصول بالضبط

### 4. بيان الـ IPs الجديدة
- 🆕 **IP جديد** - يظهر فقط للـ IPs التي تزور للمرة الأولى

## 📈 حالات الاستخدام

### 1. مراقبة الزوار:
- متابعة من يزور موقعك
- معرفة أنواع الأجهزة والمتصفحات المستخدمة
- اكتشاف الأنماط

### 2. الأمان:
- تتبع الوصول غير المصرح به
- اكتشاف محاولات الاختراق
- تتبع مصدر الهجمات

### 3. التسويق:
- معرفة أين يأتي الزوار
- معرفة أنواع الأجهزة المستخدمة
- تحسين التوافق مع الأجهزة المختلفة

### 4. التحليلات:
- عدد الزوار الفريدين
- أكثر المتصفحات استخداماً
- أنظمة التشغيل المستخدمة

---

## 🔒 الأمان

### ✅ معايير الأمان:
- معلومات الجهاز لا تُحفظ في قاعدة بيانات دائمة (ذاكرة مؤقتة فقط)
- API الزوار محمي بكلمة مرور
- التنبيهات تُرسل عبر HTTPS
- التسجيل محفوظ في ملفات آمنة

### ⚠️ ملاحظات:
- لا يتم تخزين معلومات شخصية صريحة
- User-Agent قد يحتوي على معلومات حساسة - يتم قص أول 80 حرف فقط
- IP يُستخدم فقط للأمان والتتبع
